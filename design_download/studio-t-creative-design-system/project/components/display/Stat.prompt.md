Big-number metric for results / proof bands, paired with a caption.

```jsx
<Stat value="250%" label="Increase in brand awareness for clients on our system" tone="accent" />
```

`tone`: `accent` (terracotta), `ink`, `rose`, or `cream` (on dark). `align="center"` for centered bands. Pair 3–4 in a flex row for a stats strip.
