Warm capsule call-to-action button — use for any primary or secondary action across Studio T surfaces.

```jsx
<Button variant="primary" size="lg" rightIcon={<span>→</span>}>
  Get a Custom Proposal
</Button>
```

Variants: `primary` (terracotta fill, brand shadow — one per view), `secondary` (terracotta outline), `soft` (blush fill), `ghost` (text only), `inverse` (cream fill, for dark/photo sections). Sizes `sm | md | lg`; `shape="rounded"` swaps the pill for a 12px radius. Pass `as="a"` with `href` to render a link-styled CTA.
